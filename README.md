***IZDEVUMU IZSEKOTĀJS***



Šī lietotne ir personīgo izdevumu izsekotāja programma, kas ir domāta finanšu pārvaldībai. Programma ļauj reģistrēt izdevumus, tos izsekot pa mēnešiem vai pa kategorijām, kā arī eksportēt datus uz .csv formātu, kas domāts MS excel vai google sheets.



**Uzstādīšana**



1\. Noklonējiet repozitoriju:

&#x20;  git clone \[[IZDEVUMU IZSEKOTĀJS](https://github.com/janimmelderim-svg/week_5)]

2\. Dodieties uz mapi:

&#x20;  cd week\_5

3\. Palaidiet programmu:

&#x20;  python app.py



**Lietošana**



1. Pievienot izdevumus
Ievadiet datumu (YYYY-MM-DD formāts), norādiet kategoriju no izvēles, ievada summu, pievieno aprakstu
2. Parādīt izdevumus
Funkcija parāda visu izdevumu kopsummu
3. Filtrēt pēc mēneša
Atlasiet izdevumus par katru mēnesi
4. Kopsavilkums pa kategorijām
Parāda kopsavilkumu pa ievadītajām kategorijām
5. Dzēst izdevumu
Izdezēst ierakstu
6. Eksportēt CSV
Saglabājiet datus .csv failā lietošanai MS excel vai google sheets
7. Iziet

&#x20;   Beigt lietot programmu
